# 0x02-shell_redirections

Scripts to practice Linux shell redirections and basic file handling.

## Scripts

### 0. Hello World (0-hello_world)
Print "Hello, World" followed by a new line.

### 1. Confused Smiley (1-confused_smiley)
Display a confused smiley: "(Ã”o)'".

### 2. Let's Display a File (2-hellofile)
Display the content of /etc/passwd.

### 3. What About 2? (3-twofiles)
Display the content of /etc/passwd and /etc/hosts.

### 4. Last Lines of a File (4-lastlines)
Display the last 10 lines of /etc/passwd.

### 5. I'd Prefer the First Ones, Actually (5-firstlines)
Display the first 10 lines of /etc/passwd.

### 6. Line #2 (6-third_line)
Display the third line of the file iacta (no sed allowed).

### 7. It's a Good File That Cuts Iron Without Making a Noise (7-file)
Create file '*\"Best School"\*$\?*****:)' containing "Best School".

### 8. Save Current State of Directory (8-cwd_state)
Write the result of `ls -la` into ls_cwd_content, overwrite if exists.

### 9. Duplicate the Last Line (9-duplicate_last_line)
Duplicate the last line of the file iacta.

### 10. No More Javascript (10-no_more_js)
Delete all regular files with a .js extension in current directory and subfolders.
